# Heart Failure Mortality Prediction System 🫀

This repository contains a Machine Learning project designed to predict the survival probability of patients with heart failure based on 12 clinical features. It features a trained Deep Learning model and an interactive web dashboard.

## 🚀 Features
* **Predictive Engine:** A Sequential Neural Network (TensorFlow/Keras) trained on clinical records.
* **Interactive UI:** A Streamlit-based frontend for real-time risk assessment.
* **Data Visualization:** Comparative analysis of patient metrics against historical data.
* **Clinical Mapping:** Seamlessly handles categorical data (Yes/No) for user-friendly inputs.

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **ML Frameworks:** TensorFlow, Keras, Scikit-Learn
* **Dashboard:** Streamlit
* **Data Analysis:** Pandas, NumPy, Seaborn, Matplotlib

## 📊 Dataset
The model uses the `heart_failure_clinical_records_dataset.csv`, which includes:
- **Biometric Data:** Age, Sex, Anaemia, Diabetes, High Blood Pressure, Smoking.
- **Clinical Measurements:** CPK Enzyme, Ejection Fraction, Platelets, Serum Creatinine, Serum Sodium.
- **Context:** Follow-up period (Time).
